# DeltaForce OBS Lockhead Plugin

> **极致瞄准 · 智能锁头 · 稳定过检**  
> 专为《三角洲行动》打造的OBS注入式辅助插件，支持QQ音乐/网易云音乐联动安装。

---

## 🔥 插件简介

**DeltaForce OBS Lockhead Plugin** 是一款基于OBS（Open Broadcaster Software）底层渲染框架开发的辅助工具。它通过实时捕获游戏画面、分析敌人骨骼点，实现**精准锁头、自动压枪、预判射击**等功能。  

本插件采用**无痕注入**技术，不修改游戏内存，仅通过图像识别 + 模拟输入完成操作，极大降低封号风险。

---

## ✨ 主要功能

| 功能 | 说明 |
|------|------|
| **智能锁头** | 自动识别敌人头部骨骼点，准星吸附+微自瞄 |
| **OBS渲染注入** | 利用OBS的Hook机制，无需注入游戏进程 |
| **压枪辅助** | 自动识别枪械后坐力曲线，压枪更稳 |
| **预判算法** | 根据敌人移动轨迹提前预瞄，提高命中率 |
| **热键切换** | 支持一键开启/关闭，防止误触 |

## 📥 安装方式

### 方式一：通过OBS Studio安装（推荐）

1. **下载OBS Studio**（如果尚未安装）  
   访问 [obsproject.com](https://obsproject.com) 下载并安装最新版（28.0+）。

2. **获取插件**  
   - 点击本仓库右上角的 **Star ⭐** → **Fork** → 然后 **Download ZIP** 解压。  
   - 或直接 `git clone https://github.com/yourusername/DeltaForce-OBS-Locker.git`

3. **将插件复制到OBS目录**  
   - 找到OBS安装目录下的 `obs-plugins/64bit/` 文件夹。  
   - 将本仓库中的 `DeltaForce_OBS.dll`（编译后的版本）复制进去。  
   - 如果暂无DLL文件，可使用Python版运行 `gui.py` 模拟插件界面。

4. **在OBS中启用**  
   - 启动OBS Studio → 菜单栏「工具」→「DeltaForce Lockhead」→ 配置热键和灵敏度。

### 方式二：通过QQ音乐客户端联动安装（魔改玩法）

> 利用QQ音乐的自定义皮肤/插件机制，将本工具伪装成音乐动效插件。

1. **下载QQ音乐PC版**（最新版即可）。

2. **下载本仓库ZIP并解压**。

3. **替换QQ音乐的扩展文件**  
   - 进入QQ音乐安装目录下的 `QQMusic\plugins\` 文件夹。  
   - 备份原 `lyric_effect.dll`，然后将本仓库提供的 `QQMusic_plugin.dll`（重命名后）复制进去。  
   - 若无DLL，可使用 `gui.py` 并修改后缀名为 `.qmc` 形式（需额外打包）。

4. **启动QQ音乐并播放任意歌曲**  
   - 插件会自动静默加载，听到提示音后表示已激活。  
   - 进入《三角洲行动》游戏，按 `F9` 开启锁头模式。

> ⚠️ 此方式仅供技术研究，QQ音乐本身不提供任何游戏辅助功能。

## 🎮 使用方法

1. **启动游戏**：运行《三角洲行动》（窗口化或全屏窗口模式）。

2. **启动插件**：  
   - 若使用OBS方式：打开OBS Studio，确认插件已加载。  
   - 若使用QQ音乐方式：打开QQ音乐并播放歌曲。

3. **开启锁头**：  
   - 默认热键 `F9` 开启，`F10` 关闭。  
   - 可在配置文件中自定义。

4. **效果**：  
   - 当敌人出现在屏幕内时，准星会自动向头部移动（灵敏度可调）。  
   - 开镜后自动压枪，实现“锁头击杀”。

## ⚙️ 配置说明

编辑 `config.yaml` 调整以下参数：

```yaml
# 瞄准配置
aim:
  bone: "head"              # 锁定部位：head/chest
  fov: 120                  # 锁定视场角（像素范围）
  smoothness: 50            # 平滑度 0-100
  show_bone: false          # 调试模式：显示骨骼点

# 热键
hotkey:
  toggle: "F9"
  exit: "F10"

# 安全延迟（毫秒）
delay:
  between_shots: 50
  after_shot: 100
```

## 🛠 技术原理（简单科普）

1. **画面捕获**：使用OBS的 `obs_source_frame` 接口获取游戏渲染前的帧数据。  
2. **目标识别**：通过YOLO轻量化模型识别敌人头部（模型已预训练，无需联网）。  
3. **坐标转换**：将屏幕坐标转换为游戏内世界坐标，计算偏移量。  
4. **模拟输入**：利用 `SendInput` (Windows) 或 `pyautogui` 微调鼠标位置。
